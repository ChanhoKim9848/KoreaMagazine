The Youtube video that I utilized explained how to make header, footer, buttons, div class for coding the html.

Secondly, since I wanted to design my website, I needed to use css and javascript.
when I was using css stylesheet, the youtube video showed how to link my html to css stylesheet and explain how to use internal stylesheet and external style sheet.

Lastly, after I figured out the internal stylesheet and external one, I could design my website by using css stylesheet code and even javascript one on html not on external stylesheet, allowing me to make website quickly and efficiently.
